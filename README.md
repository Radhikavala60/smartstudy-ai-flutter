# Description
SmartStudy AI is a Flutter-based application that integrates a public REST API
to fetch educational content related to computer science.

# Features
- Public REST API Integration
- JSON Parsing
- GridView UI
- Flutter & Dart
- Android Studio / VS Code

# API Used
https://openlibrary.org/subjects/computer_science.json

# Output Description
The app displays computer science books in a grid format.
Each card shows the book title and first published year.
